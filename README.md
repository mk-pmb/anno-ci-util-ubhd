anno-ci-util-ubhd
=================
<!--/#echo -->

<!--#echo json="package.json" key="description" -->
Shared CI tools for our web annotation projects.
<!--/#echo -->



Usage
=====

#### on GitHub:

Add a step like this to your CI workflow:

```text
      - uses: mk-pmb/anno-ci-util-ubhd/.github@v1
```


#### Plain shell / other CI frameworks:

* Clone this repository or install it as npm dependency.
* In the top-level directory of your project repository, run
  `…/anno-ci-util-ubhd/repo/run_all_default_checks.sh`
